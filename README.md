#  Asisten Asa

**Asisten Asa** adalah aplikasi berbasis web yang dirancang untuk membantu penyandang disabilitas dalam menjalani aktivitas digital dengan lebih mudah, mandiri, dan inklusif.

Project ini dikembangkan untuk mendukung aksesibilitas teknologi dan memberikan solusi praktis bagi berbagai kebutuhan pengguna disabilitas.

---

#  Tujuan Project

Tujuan utama dari **Asisten Asa** adalah:

* Membantu pengguna disabilitas mengakses teknologi dengan lebih mudah
* Menyediakan fitur aksesibilitas yang praktis
* Mendukung inklusivitas dalam dunia digital
* Memberikan pengalaman penggunaan yang ramah bagi semua orang

---

# ✨ Fitur Utama

### 🗣️ Voice Assistant

Pengguna dapat mengontrol aplikasi menggunakan perintah suara.

### 🔎 Deteksi Uang

Membantu pengguna dengan keterbatasan penglihatan untuk membaca nominal uang berbasis machine learning.

### 🎨 Accessibility Mode

Mode tampilan khusus untuk meningkatkan keterbacaan seperti:

* High contrast
* Font besar
* Mode ramah disleksia

### 🧭 Simple Navigation

Navigasi sederhana yang mudah digunakan oleh semua kalangan.

# ❤️ Misi Kami

Kami percaya bahwa **teknologi harus dapat diakses oleh semua orang**, tanpa terkecuali.

Asisten Asa dibuat untuk menjadi langkah kecil menuju dunia digital yang lebih inklusif.
